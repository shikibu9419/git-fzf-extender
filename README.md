# Git fzf Extender

A git extender plugin depending on [fzf](https://github.com/junegunn/fzf).

## Installation

### zplug

```shell
zplug 'shikibu9419/git-fzf-extender', use:init.zsh
```

## Commands
add, checkout, cherrypick, stash, reset and worktree
